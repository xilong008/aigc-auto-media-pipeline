## 核心逻辑：用"血淋淋的败诉案例"和"省了多少钱"替代法条堆砌

❌ 错误："劳动合同签订时要注意各项条款，保护自己的权益。"
✅ 正确："上个月刚帮一个小老板打完仲裁，对方员工以'未缴社保'为由申请2N赔偿，本来要赔18万。但我翻了他的微信聊天记录，发现员工自己书面承诺过放弃社保——最终仲裁裁定公司只需补缴社保差额2.3万，省了15.7万。这种'反杀'机会，90%的老板根本不知道。"

## AIDA 定制版

### 1. Attention (15%) — 利益恐惧切入
- 角度：不懂法被坑的金额、错过时效的遗憾、"以为没事结果被告"
- *示例*："90%的小老板不知道，员工主动离职也能告你赔2N，关键就看一份文件签没签。"

### 2. Interest (30%) — 真实案例 + 金额量化
- 必含：涉案金额、法条编号、时效期限、成功率数据
- **术语库**：2N赔偿、经济补偿金、N+1、劳动仲裁、诉讼时效、举证责任倒置、竞业限制、商业秘密、股权代持、阴阳合同、税务稽查、核定征收、查账征收、小规模/一般纳税人、进项抵扣、发票风险、关联交易、实缴/认缴、工商年报、知识产权贯标、专利无效宣告
- *示例*："别不信，同样是公司注销，走简易注销15天搞定花800块，走普通注销可能拖6个月花1.5万。区别就在于你有没有欠税记录和未结诉讼。但90%的代办不会告诉你先去税务局拉一张'完税证明'做预检。"

### 3. Desire (40%) — "拿来就能用"的法律/财税操作
- 具体到"存什么证据"、"签什么条款"、"找谁办什么手续"
- *示例*："小老板自保三件套：1.所有员工入职当天签《劳动合同》+《保密协议》+《竞业限制协议》，三份缺一不可；2.辞退员工必须书面送达+EMS备份+签收回执；3.每月工资必须银行转账而非现金，备注写清'XX月工资'，打官司时这就是铁证。"

### 4. Action (15%)
- *示例*："遇到法律/税务问题拿不准？评论区留下【你的情况+行业】，我帮你初步判断风险等级。"
