## 行业核心逻辑：用真实的"术后翻车"和"价格黑幕"替代一切模糊承诺

*   ❌ **错误示范**："做了热玛吉之后皮肤变得很好，大家可以试试，效果很明显。"
*   ✅ **正确示范**："说句不好听的，我见过最惨的姐妹，在某连锁店花了1.2万做热玛吉，结果用的是400发的探头打了全脸，能量参数只开到2.0，根本到不了真皮层。3个月后一点效果没有，去正规机构复查才知道，至少要600发、能量3.5以上才有胶原再生效果。"

## AIDA 结构定制版

### 1. Attention (15%) — 颜值焦虑 + 价格恐惧切入
*   **切入角度**：花了大价钱没效果的焦虑、被割韭菜的恐惧、对"正品"vs"水货"的不安全感
*   *示例*："医美行业最大的坑不是贵，而是你花了钱根本不知道打进去的是什么。今天我把这层窗户纸彻底捅破。"

### 2. Interest (30%) — 量化数据 + 行业黑话建立信任
*   **必须包含**：项目价格区间、仪器参数、有效成分浓度、恢复周期天数
*   **行业专业术语库**：真皮层、胶原再生、SMAS筋膜层、玻尿酸交联度、肉毒素单位(U)、光电项目、IPL/DPL、皮秒/超皮秒、点阵激光、水光针、非交联玻尿酸、术后红肿期、代谢周期、院线级、械字号/妆字号、CE/FDA认证、超声刀频率、脉冲宽度、能量密度(J/cm²)
*   *示例*："别不信，同样叫'水光针'，有的机构用的是3ml非交联玻尿酸兑了大量生理盐水，成本不到200块，收你2800；真正有效的应该是含2ml交联玻尿酸+谷胱甘肽的定制配方，成本就要800+。"

### 3. Desire (40%) — "到店就能用"的避坑指南
*   **要求**：具体到"问什么问题"、"看什么标签"、"要求出示什么证件"
*   *示例*："做任何光电项目前，必须要求三件事：1. 看仪器铭牌上的FDA/CE编号是否能在官网查到；2. 要求当面拆封耗材包装，核对批号和有效期；3. 术前拍照存档，约定好术后复查时间。"

### 4. Action (15%) — 医美行业专属CTA
*   *示例*："想做但拿不准项目和价格？评论区留下【你的诉求+预算】，我帮你拆解到底值不值得做。"

## 配图提示词风格指引

医美行业图片应聚焦以下视觉方向：
1. **封面图**：干净明亮的皮肤特写 / 精致的护肤场景
2. **科普图**：皮肤层次结构示意 / 仪器设备特写
3. **对比图**：术前术后效果对比（注意合规性）
4. **氛围图**：高端诊所环境 / 专业医生操作场景

**风格关键词**：flawless glowing skin closeup, clean clinical aesthetic, soft diffused lighting, luxury skincare flatlay, medical spa interior, professional beauty photography, pastel tones
